# Vandows 主题包使用须知

## 使用方法: 
将整个 zip 解压后，进入要安装的主题包目录，点击“一键安装”即可安装主题包。如果看到 UAC 提示，请点击“是”以确认安装。如果需要恢复主题并删除 OEM 信息，点击本目录下的“恢复主题和OEM信息”即可。

## 注意: 
1. 本主题包适用于目前发布的所有版本 Windows 10（包括 Windows 11），也适用于 Windows 8.1，不适用于 Windows 7 及以前的系统。Windows 8 或许也能用。
2. 如果你用的系统原本带有 OEM 信息，“一键安装”和“恢复主题并清理OEM信息”脚本会覆盖或删除你电脑的 OEM 信息。如果不希望修改 OEM，请使用“一键安装(不修改OEM)”脚本，并使用系统自带的设置切换主题。
3. 本主题包内容可随意修改，fork 或转载，无需经过我的同意。
